## Phase 1: Containment

1. Immediately isolate the affected host from the network.
2. Do **not** power down the machine (preserve RAM for forensics).
3. Disable compromised Active Directory accounts.
4. Notify the Incident Response Team and escalate to management.
5. Document all actions taken and observed symptoms for later analysis.
