# 🏗️ The Sandbox

This is my developer playground built with **Astro**. This repo is automatically deployed to GitHub Pages via GitHub Actions.

## 🚀 Live Site

[View the playground here](https://dfirsec.github.io/the-sandbox/)

## 🧪 Experiments

- [ ] **Blog:** Markdown-based content collections.
- [ ] **Portfolio:** Project showcase section.
- [ ] **React Island:** Theme toggle / Interactive gallery.

## 🛠️ Local Setup

1. Clone the repo.
2. Run `npm install`.
3. Run `npm run dev`.
